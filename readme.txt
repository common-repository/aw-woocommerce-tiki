Woocommerce TIKI adalah Plugin khusus Woocommerce yang digunakan untuk mengintegrasikan ongkos kirim dengan total belanja calon konsumen Anda.

== Description ==

Woocommerce TIKI Shipping mengintegrasikan ongkos pengiriman dengan total belanja konsumen Anda pada saat checkout. Dengan beragam fitur yang ditawarkan, plugin ini sungguh powerful dan menjadikan plugin ini wajib digunakan ke semua toko online yang menggunakan wordpress.

= WooCommerce TIKI Free Version =

* Ongkos kirim TIKI Regular
* Data kota TIKI Jakarta ke Seluruh Indonesia

= WooCommerce TIKI Full Version =

* Ongkir kirim TIKI REG, ONS, ECO, HDS, dan SDS
* Data kota TIKI dari basis kota pengiriman yang dipilih ke Seluruh Indonesia
* Checkout Provinsi, Kota, Kecamatan saling terkait
* Biaya Tambahan ( untuk Packing dll )
* Fitur Kalkulasi Ongkir di Halaman Cart
* Gratis update dan Bantuan Support

= WooCommerce TIKI Exclusive Version =

* Ongkir kirim TIKI REG, ONS, ECO, HDS, dan SDS
* Data kota TIKI dari basis kota pengiriman yang dipilih ke Seluruh Indonesia
* Checkout Provinsi, Kota, Kecamatan saling terkait
* Cek Ongkir di halaman produk
* Cek Ongkir Shortcode
* Cek Ongkir Widget
* Tracking mengirimkan no resi langsung ke email pelanggan. ( Lacak Pengiriman )
* Tracking shortcode ( Lacak Pengiriman )
* Tracking widget ( Lacak Pengiriman )
* Free Shipping (Pengiriman Gratis)
* Kalkulasi Ongkir di Halaman Cart
* Gratis update dan Bantuan Support

== Installation ==

Dibawah ini adalah 3 cara mudah untuk melakukan instalasi pada plugin WooCommerce Kode Pembayaran

1. Untuk menginstal plugin caranya cukup mudah. Apabila Anda ingin mengupload langsung menggunakan file zip. Melalui wordpress pada menu Plugins > Add New > Upload Plugin
2. Aktifkan Plugin
3. Halaman pengaturan plugin di WooCommerce > Settings > TIKI Shipping
4. Anda sudah bisa menggunakan plugin ini, serta dapat mengkonfigurasi plugin sesuai dengan keinginan Anda.
5. Have fun!
